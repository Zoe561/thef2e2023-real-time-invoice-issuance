# 🚧 歷年總統開票地圖 施工中 🚧

## 作品說明

The F2E 2023 mission2 主題，利用網頁技術將歷年總統大選的結果呈現於地圖上。
The F2E 2023 官網： <https://2023.thef2e.com/>
設計師： jhen → <https://2023.thef2e.com/users/12061579704041679194>
網頁工程師：Zoe

## 系統說明

使用 Angular 16 開發，專案運行方式

1. `npm install`
2. `npm start`
